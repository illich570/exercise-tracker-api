Exercise tracker API

Made with Express, and Mongoose

Casos de uso: primero registrar un usuario

Despues con el ID del usuario, registras ejercicios mediante info

Form to register a user
Form for track exercise

Ejemplo de consulta: http://localhost:8080/api/exercise/log?userId=1

Parametros: { } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

/api/exercise/log?{userId}[&from][&to][&limit]

Url-shortener made by @IllichR
